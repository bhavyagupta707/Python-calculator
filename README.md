# Python Calculator

A simple calculator built using Python. It performs basic and advanced mathematical operations such as Addition, Subtraction, Multiplication, Division, Power, Percentage, and Square Root.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Power (^)
- Percentage (%)
- Square Root (√)
- Handles division by zero errors

## Technologies Used

- Python 3
- Math Module
